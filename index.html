Since COVID-19 has been spreading widely around the world and becomes a severe problem for many countries, it is meaningful to monitor the situations on a daily basis. Therefore, I plan to present a world map with different number of confirmed casees in each country. Additionally two line charts will be applied to compare the circumstances in China and outside China.

Click / touch a circle for more information.

Sources: compilation of Johns-Hopkins University updated on March 27, and based on data from the WHO, CDC, NHC and DXY.cn

Since 03.27.2020, The COVID-19 has caused 30438 deaths and more than 650000 confirmed infections worldwide, according to figures compiled by Johns-Hopkins University. Given line charts above, it is obvious that China has more than 73 times as confirmed cases as in other countries in January. However, COVID-19 has become a widely spread case two months later and many countries have more than 10000 confirmed cases apart from China and this trend will keep going. The limitation of my approach is that it cannot realize a real-time dashboard without using an API to JHU databases. I will try to make it in the future.

In conclusion, I manage to achieve a static dashboard showing worldwide situation of COVID-19 as well as two line charts illustrating a comparison between China and other countries. To improve the website, I will try to learn how to use API to achieve a real-time dashboard.
